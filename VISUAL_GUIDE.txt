    ❌ "pip is not recognized"
    ───────────────────────────────────
    ➜ Python chưa cài
    ➜ Tải: https://www.python.org/downloads/
    ➜ NHỚ TICK: "Add Python to PATH"

    ❌ "Invalid API key"
    ───────────────────────────────────
    ➜ Mở lại file .env
    ➜ Check key có đúng không
    ➜ Không có dấu cách thừa

    ❌ "Port 8501 already in use"
    ───────────────────────────────────
    ➜ Đóng app cũ
    ➜ Hoặc chạy: streamlit run app.py --server.port 8502
