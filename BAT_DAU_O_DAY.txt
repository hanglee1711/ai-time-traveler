⚡ MUỐN CHẠY CỰC NHANH? LÀM 3 VIỆC NÀY:

    1️⃣  Lấy API key: https://aistudio.google.com/app/apikey
         (Đăng nhập Gmail → Create API Key → Copy)

    2️⃣  Mở file .env bằng Notepad, gõ:

         AI_PROVIDER=gemini
         GEMINI_API_KEY=dán-key-vào-đây
         GEMINI_MODEL=gemini-1.5-pro

         Lưu lại!

    3️⃣  Double-click: CHAY_NGAY.bat

    🎉 XONG!
